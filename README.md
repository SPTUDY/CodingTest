# 📘 Coding Test Study

## 🔎 진행 방식
- 매주 정해진 섹션에 대한 공부를 `[member]-day[n]` 브랜치로 푸시
    - 📝 : `[member]/[date]/[problem_number].java`
    - 문제에 따른 강의는 알아서 수강
- 기한: 각 day+1일 오전 9시까지
- 벌칙: push 안되어있으면 10,000원 벌금
